# Glassmorphism Card Creator

A modern Glassmorphism Card Creator built with HTML, CSS, and Vanilla JavaScript. Customize blur, opacity, and border radius with live preview and copy the generated CSS instantly.

## ✨ Features

- Live Glassmorphism Preview
- Blur Control
- Background Opacity Control
- Border Radius Control
- Auto Generated CSS Code
- One Click Copy CSS
- Responsive Design
- Modern UI
- Pure HTML, CSS & JavaScript
- No External Libraries

## 📂 Project Structure

```
glassmorphism-card-creator/
│── index.html
│── style.css
└── app.js
```

## 🚀 Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## 📖 How to Use

1. Open `index.html` in your browser.
2. Adjust the Blur slider.
3. Change the Opacity level.
4. Set the Border Radius.
5. Copy the generated CSS with one click.
6. Paste the CSS into your own project.
